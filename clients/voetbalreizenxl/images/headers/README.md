# Headerafbeeldingen VoetbalreizenXL

Headerafbeeldingen bovenaan de nieuwsbrief, per campagnethema.

## Vereisten
- Formaat: **590px breed**, aanbevolen hoogte 200-300px (liggend)
- Bestandstype: PNG of JPG
- Naamgeving: thema in lowercase met koppeltekens, bijv. `serie-a-2026.png`

## Werkwijze
1. Afbeelding hier opslaan
2. Uploaden naar Brevo-beeldbibliotheek
3. URL invullen als `header_image_url` in het campagne-script, of als vaste waarde in `voetbalreizenxl.json`

## Huidige header
De standaard headerafbeelding staat als `header_image_url` in `templates/brand/voetbalreizenxl.json`.
Thema-specifieke headers kunnen per campagne worden overschreven in het generatiescript.
